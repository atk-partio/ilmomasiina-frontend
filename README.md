ilmomasiina-frontend
====================

Requirements
------------
* [Node.js](http://nodejs.org/download/) >= 0.8
* [npm](https://npmjs.org)

Installing
----------
```bash
npm install
```

Building
--------
```bash
./gulp.sh
```

Run tests
---------
Run the Karma runner
```bash
./node_modules/.bin/karma start
```
